# Проект Dash для анализа данных о количестве солнечной радиации

## Описание

Этот проект представляет собой веб-приложение на Dash, разработанное для анализа данных солнечной радиации. Приложение позволяет загружать данные из CSV-файлов, отображает графики для GHI, DNI и DHI

## Установка

Для запуска этого проекта вам понадобится Python версии 3.6 или выше. Следуйте этим шагам для установки и запуска приложения:

1. Установите необходимые библиотеки, используя pip:
pip install dash
pip install pandas
pip install plotly graph_objects

2. Запустите приложение, выполнив следующую команду в терминале:
python dashboard.py

Приложение будет доступно по адресу http://127.0.0.1:8050.

## Использование

После запуска приложения перейдите по адресу http://127.0.0.1:8050 в вашем браузере. Вы увидите интерфейс приложения и параметры для анализа.

## Зависимости

- Dash
- Pandas
- Plotly
